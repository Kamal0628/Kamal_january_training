# Employee Performance Prediction

## Problem Statement
Predict employee performance score using supervised machine learning algorithms.

## Dataset Description
Dataset contains employee details such as age, salary, experience, department, and performance score.

## Data Cleaning Steps
- Removed duplicates
- Encoded categorical variables
- Removed irrelevant ID column
- Treated outliers using IQR
- Applied feature scaling

## Algorithms Used
- Linear Regression
- Decision Tree
- Random Forest
- KNN
- SVM

## Evaluation Metrics
- R2 Score
- MSE
- RMSE
- MAE

## Conclusion
Random Forest performed best with highest R2 score.
